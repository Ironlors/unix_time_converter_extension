# unix_time_converter_extension
This repository contains all the files to create the unix_time_converter extension from my blog post.
The folder structure is the following.

```
dist
├── index.html
├── manifest.json
└── res
    ├── css
    │   ├── all.min.css
    │   ├── bootstrap.min.css
    │   └── style.css
    ├── img
    │   ├── icon128.png
    │   ├── icon16.png
    │   └── icon48.png
    ├── js
    │   ├── bootstrap.bundle.min.js
    │   ├── jquery.min.js
    │   └── popup.js
    └── webfonts
        ├── fa-brands-400.eot
        ├── fa-brands-400.svg
        ├── fa-brands-400.ttf
        ├── fa-brands-400.woff
        ├── fa-brands-400.woff2
        ├── fa-regular-400.eot
        ├── fa-regular-400.svg
        ├── fa-regular-400.ttf
        ├── fa-regular-400.woff
        ├── fa-regular-400.woff2
        ├── fa-solid-900.eot
        ├── fa-solid-900.svg
        ├── fa-solid-900.ttf
        ├── fa-solid-900.woff
        └── fa-solid-900.woff2
```
